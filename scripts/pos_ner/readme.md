# Intro

The files in this repo can be used to tag text corpora with parts-of-speech (POS) or named-entity (NER) information. When possible, different scripts use different tools such as the Natural Language Toolkit (NLTK), spaCy, and Stanza have been used to give users flexibility in the format and focus of their outputs. You should choose the one that works best for your pipeline. You can contact [bcritt@stanford.edu](mailto:bcritt@stanford.edu) for support if you aren't sure which script fits your workflow.

The jupyter notebooke (.ipynb) can be run with Sherlock's OnDemand functionality. All relevant script fragments are included in the ipynb file since interactive computing will give you greater control over which parts of the process you want to run.
