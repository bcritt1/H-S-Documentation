# Intro

The files in this repo can be used to tag text corpora with parts-of-speech (POS) or named-entity (NER) information. In python, different scripts use different tools such as the Natural Language Toolkit 
[(NLTK)](https://www.nltk.org/), [spaCy](https://spacy.io), and [huggingface](https://huggingface.co/) have been used to give users flexibility in the format and focus of their outputs. The R script uses 
only spaCy due to ease of use. You should choose the one that works best for your pipeline. You can contact [bcritt@stanford.edu](mailto:bcritt@stanford.edu) for support if you aren't sure which script fits your workflow.
