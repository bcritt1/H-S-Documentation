# Ipynbs

A collection of Ipython notebooks, intended for use with Jupyter Notebooks, made available on Sherlock through [Open OnDemand](https://login.sherlock.stanford.edu/). The notebooks themselves are 
heavily-documented and in most cases are adaptations of batch scripts available in the other repos here. In other words, use these scripts if you'd like to run jobs interactively, and the batch scripts if 
that's not necessary. [map.ipynb](/scripts/ipynbs/map.ipynb) represents a special case because its sole purpose is to create an interactive map, functionality that isn't replicable in a batch environment..
